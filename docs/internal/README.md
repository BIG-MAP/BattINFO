# Internal / Maintainer Notes

This directory holds **maintainer-facing working notes** — design drafts, migration
runbooks, status trackers, and integration references — that are intentionally
**not** part of the rendered documentation site (they are excluded in `../conf.py`).

They are kept in the repository for contributors and for historical context, but
they are not user-facing guides and may use superseded terminology.

For user documentation, build the docs site from `docs/` or start at
[`../index.md`](../index.md).
