# Workspace Examples

These examples use the disk-backed BattINFO workspace MVP:

- `battinfo-workspace.json` declares the workspace contract and registry target.
- `resources/cell.json`, `resources/test.json`, and `resources/dataset.json` are the editable authoring resources.
- `artifacts/` holds local files referenced by the dataset resource.
- `dist/` is generated by `battinfo workspace validate` and `battinfo workspace bundle` and is not tracked as source of truth.

`digibatt-cr2032-release/` is the reference DigiBatt-style happy-path example for
`battinfo workspace init`, `battinfo workspace validate`, and
`battinfo workspace bundle`.
