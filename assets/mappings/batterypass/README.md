# Battery Pass Mapping (v1.2.0)

This folder will contain alignment and mapping artifacts to transform BattINFO
canonical JSON into Battery Pass JSON-LD. The mapping is pinned to
BatteryPassDataModel v1.2.0.

Planned artifacts
- `alignment.ttl` semantic alignment between domain-battery and Battery Pass
- `rules/` transformation rules (JSON -> JSON-LD)
- `profile/` constraints specific to Battery Pass subsets
