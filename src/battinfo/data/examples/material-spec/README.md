# Material Specs (Examples)

Reusable material specifications — the *spec* half of the `material-spec` + `material`
spec/instance pair. Each file describes a battery material grade (active material,
binder, conductive additive, electrolyte salt/solvent, metal electrode) that cell and
component specs can reference by IRI.

Coverage: graphite, LFP, NMC811, NMC622, LCO, LMFP, LNMO, zinc, carbon black, PVDF,
KOH, LiPF6, EC, EMC. Cathode/anode actives and the LiPF6/EC/EMC electrolyte
constituents are grounded in the DIGIBAT Discovery-Benchmark coin-cell corpus
(Canrud-supplied); LNMO, zinc and KOH are synthetic reference examples.
