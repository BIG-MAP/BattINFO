# BattINFO guide notebooks

The guide notebooks live in [`docs/guides/`](../../docs/guides/).

| Notebook | What it covers |
|---|---|
| [01-concepts.ipynb](../../docs/guides/01-concepts.ipynb) | Data model, record types, IRIs, the semantic layer |
| [02-first-cell-type.ipynb](../../docs/guides/02-first-cell-type.ipynb) | Materials → components → cell type → publish |
| [03-linked-records.ipynb](../../docs/guides/03-linked-records.ipynb) | Cell instance → test → dataset → publication package |
| [04-semantic-layer.ipynb](../../docs/guides/04-semantic-layer.ipynb) | JSON-LD anatomy, EMMO type stacking, RDF and SPARQL |
| [05-descriptors.ipynb](../../docs/guides/05-descriptors.ipynb) | Research-grade descriptors: electrodes, electrolyte, separator |

Open notebooks from the **repo root** with the `.venv` kernel selected.

Notebooks write temporary data under `.battinfo/notebooks/` and do not modify curated repository examples.

The `legacy/` subdirectory contains the earlier alpha walkthrough notebooks, retained for reference.
