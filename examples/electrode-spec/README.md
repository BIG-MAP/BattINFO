# Electrode Specs (Examples)

Reusable electrode specifications — the *spec* half of the `electrode-spec` + `electrode`
pair. Each describes a coating (active material + binder + conductive additive, referencing
standalone `material-spec` records by IRI) on a current collector. Grounded in the
DIGIBAT Discovery-Benchmark coin cells (NMC811 cathode, graphite anode; Canrud).
